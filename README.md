# wanman

Agent Matrix framework — open-source runtime, CLI, and SDK for autonomous agent collaboration. Runs a supervised network of Claude/Codex agents on your machine, coordinated through a JSON-RPC supervisor and sandboxed via `@sandbank.dev/boxlite`.

For the hosted multi-tenant product (web dashboard, launch/story orchestration, GitHub App integration, billing), see [wanman.ai](https://wanman.ai).
